# ⚡️프로젝트 개요
"SixhundredBills" 프로젝트는 A6조의 팀 프로젝트로,  
사용자 인증, 프로필 관리, 게시물 CRUD, 댓글 및 대댓글 CRUD, 좋아요 기능을 구현한 웹 애플리케이션입니다!!


## 💡기술 스택

- **백엔드**
  - **Spring Boot**
  - **Java**
  - **MySQL**
  - **JWT (JSON Web Token)**
  - **Hybernate**

- **프론트엔드**
  - **Figma**
  - **Postman**

- **보안**
  - **Spring Security**
  - **JWT (JSON Web Token)**
